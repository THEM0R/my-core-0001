# My Core 0001

Simple PHP core project.

## Description

This project is a simple core structure written in PHP.
It is designed as a base for building web applications using MVC architecture.

## Project Structure

```
my-core-0001
│
├── engine/
│   ├── core/
│   ├── config/
│
├── public/
│
└── index.php
```

## Requirements

* PHP 8+
* Apache / Nginx
* MySQL / MariaDB

## Installation

1. Clone the repository

```
git clone https://github.com/your-username/my-core-0001.git
```

2. Move the project to your local server directory
   Example for OpenServer:

```
domains/my-core-0001
```

3. Configure the database in

```
engine/config/config.php
```

4. Run the project in the browser

```
http://my-core-0001.local
```

## Features

* Simple routing
* MVC structure
* Database connection
* Modular architecture
